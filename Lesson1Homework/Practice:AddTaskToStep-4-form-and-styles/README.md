# Step 4 — Form and styles (final)

Adds the task form and a linked stylesheet. This is the canonical, finished version of the Lesson 1 project.

## Run it

Open `index.html` directly in a browser.

## Talking points

- Point out `form`, `action`, `method`, `label`, and the submit `button`.
- Follow the in-page “Skip to today’s tasks” link using both the mouse and keyboard.
- Submit the form and ask what happened — the browser navigates, but nothing was saved.
- Make one small CSS edit live (color or spacing) and refresh to show the edit → save → refresh loop.
- This is the motivation for Lesson 2: the page needs to be generated by a program that can remember things.
